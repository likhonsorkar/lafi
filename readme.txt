== Description ==

A sleek, modern WordPress theme with powerful customization options for a personalized and responsive website

== Installation ==

1. Upload the `theme-name` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance' menu in WordPress.
3. Customize the theme settings by going to the 'Customizer' section in the WordPress admin panel.
4. Create and assign menus to the available menu locations.
5. Configure the widgets in the 'Widgets' section.
6. Set up your homepage by creating a new page and assigning it as the 'Front page' in the 'Settings' > 'Reading' section.

== Features ==

- Responsive layout for optimal viewing on different devices.
- Customizable header and footer , widget.
- Integrated with Bootstrap framework for enhanced styling and functionality.
- Custom widgets for displaying sidebar and footer.
